PIP IN ADOBE PREMIERE PRO

How to do a picture-in-picture edit using Adobe Premiere Pro

Ever wondered how you can create a video where one clip is playing in the background while another plays on top of it? Widely known as the picture-in-picture effect, using tis technique makes it that much easier to get a point across as we can use whatever visual aid we need in the background as we explain it. Let's take a look at how we can go about achieving this look without much of a hassle.

Create a footage card

1. Use the effects contorl panel to scale and position the adjust the general view of the desired clip.


2. Place the clip that needs to be made a hole through above the the adjusted clip.


3. Resize and alter the position of the clip so that it fits and the important items are visible throughout.


4. Make sure the required part of the clip is being used.

Add a border

1. Create a text layer between the two footage layers in the timeline.


2. Delete the text inside it using the essential graphics panel.


3. Click on the page button and add the corresponding shape to the view.


4. Resize it to be just slightly larger than the footage card, which will give it the look of a border.

Nest all the clips into one unit

1. Hightlight both the clips along with the text layer, which now contains the border. Then right-click and select nest.


2. Now if you need to implement any animations or effects to the clips, you can add them directly to the nested unit without appying them one by one to each clip.

Make it full screen

1. Create a 5 seconds long color matte with the same resolution.


2. Add two keyframes at the begining for position and size.


3. Move to the end of the color matte and adjust the position and the size to the desired measurements.


Save it as a preset for later reference.

Summery

Use these general steps and create an easy picture-in-picture effect within your edit. You can tweak the appearance of it by simply playing around with the effects control panels of the text layer and the footage card. BUT always make sure to are happy with how it looks before nesting the layers.
